My Workflow

I rely on Jade and Sass for outputing my HTML and CSS using Prepros. My current text editor is Sublime Text and I use Git through Source Tree to backup my work on Bitbucket.

I tend to favor hand crafted sites over Wordpress ones due to development speed and performance. Naturally, I opt for the right tool for the job so if I find that the client needs a Content Management System, I'm more than happy to develop with it.

When I'm getting creative I use Adobe's suite of apps including Illustrator, Photoshop, InDesign, After Effects and Premiere Pro.
